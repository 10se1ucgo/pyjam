pyjam

pyjam is an open source, cross-platform audio player for Source and GoldSrc engine based games, written in Python

REQUIREMENTS

  • Tested on CPython 3.5. 2.7+ should work.
  • wxPython Phoenix - GUI
    • Compiling for Python 3.5 is a bit broken. Patch + a wheel for CPython 3.5 x64 can be found here
  • ObjectListView - wx.ListCtrl wrapper (much easier to use)
  • watchdog - Cross-platform file system monitoring

Backwards compatibility

  • shutilwhich - Backport of shutil.which() for Python <3.3
